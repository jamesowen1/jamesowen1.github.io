Pacman

For my first project at University I was tasked with recreating the original Pacman game. For this I used the SDL framework provided by my University and welcomed the challenge of making it as close to the original game as I could.

As this was my first project using C++ I feel like the results are fairly nice. I had a complete Pacman game that ran cleanly and everything worked as it should.

There is a lot that I could improve on as I was mainly focusing on learning the language and framework throughout this project. It would need an overall clean up and some errors could be made more efficient.
